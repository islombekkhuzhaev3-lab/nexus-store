## 📋 Батафсилотлар

**Antigravity Hub** — интернет-дўкон, социал платформа ва музыка плеерни биргаликда жамъ қилган инновацион эко-система.

### ✨ Асосий хусусиятлари

- 🎵 **Интерактив Музыка Плеер** - 5 та мусиқа, прогресс-бар, плейлист
- 🛍️ **Маркетплейс** - 6 та премиум маҳсулот ҳимояси ва аксессуарлари
- 📱 **Социал Эълонлар** - Форма ва модерация системаси
- 🎨 **Ду Мавзу** - Cyberpunk Neon ва Minimalist Luxury
- 💳 **Харид Савати** - Полная функциональность
- 📱 **Мобиль Симулятор** - Реалистичный прототип

### 🛠️ Технологиялар

- HTML5 / CSS3 / JavaScript
- Lucide Icons
- Responsive Design

#### Мобиль Версия (React Native)

- Lucide React Native

## 🚀 Ишлатишни Бошлаш

```bash

# Android uchun
```
